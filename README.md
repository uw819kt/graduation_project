# README

## 開発言語

* Ruby 3.3.0
* Rails7.2.2
 
## 実行手順
``` bash
$ git clone git@github.com:uw819kt/graduation_project.git
$ cd graduation_project
$ bundle install
$ rails db:create && rails db:migrate
$ rails s
```
